💡 Cas d'usage courants

  • Headers HTTP Authorization: Basic base64(user:password)
  • Images inline dans du HTML/CSS: data:image/png;base64,...
  • Tokens JWT (chaque partie est en Base64 URL-safe)
  • Transmission de données binaires dans du JSON
  • Variables d'environnement pour les secrets
